# Global-Superstore-Sales-Dashboard 


## 📌 Overview

This project presents an interactive **Power BI dashboard** built using the Global Superstore dataset to analyze sales performance, profitability, and customer behavior across multiple regions and product categories.

The objective is to transform raw business data into actionable insights for better decision-making in sales, marketing, and inventory management.

---

## 🚀 Key Features

* 📊 Interactive dashboard with dynamic filtering and drill-down capabilities
* 📈 Real-time tracking of key performance indicators (KPIs)
* 🌍 Region-wise and category-wise performance analysis
* 📉 Trend analysis across time for sales and profit

---

## 📊 Key Metrics

* Total Sales
* Total Profit
* Profit Margin
* Year-over-Year (YoY) Growth
* Category & Sub-category Sales
* Regional Performance
* Repeat Customer Rate

---

## 🛠️ Tech Stack

* **Power BI**
* **DAX (Data Analysis Expressions)**
* **Excel / CSV Dataset**

---

## 📂 Project Workflow

1. Data Cleaning and Preparation
2. Data Modeling in Power BI
3. Creation of DAX Measures:

   * Profit Margin
   * YoY Growth
   * Customer Metrics
4. Dashboard Design and Visualization
5. Insight Extraction

---

## 📊 Key Insights

* Identified top-performing regions contributing the highest share of revenue
* Highlighted product categories with strong sales but lower profit margins
* Observed seasonal trends impacting sales and profitability
* Analyzed customer behavior to identify repeat purchasing patterns
